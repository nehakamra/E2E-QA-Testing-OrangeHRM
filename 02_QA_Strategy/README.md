# QA Strategy

## Test Types
- Smoke Testing
- Functional Testing
- Regression Testing
- API Testing
- UAT Support

## Test Approach
- Risk-based testing
- Business-critical flows tested first
- Manual testing for UI
- API testing for backend validation

## Entry Criteria
- Stable build deployed
- Test environment available

## Exit Criteria
- 95% test cases executed
- No open Critical or High defects

# End-to-End QA Testing Project – OrangeHRM

## Project Objective
This project demonstrates end-to-end QA ownership for a real web application,
covering QA strategy, test planning, execution, defect management, and release sign-off.

## Application Under Test
OrangeHRM Open Source Demo Application  
URL: https://opensource-demo.orangehrmlive.com  
Credentials: Admin / admin123

## QA Scope
- Requirement analysis
- Test strategy and planning
- Manual testing
- API testing
- Test case management using Azure DevOps (ADO)
- Defect management
- QA metrics and release sign-off

## Tools & Techniques
- Manual Testing
- Postman (API Testing)
- Azure DevOps (Test Case Management – Process)
- Risk-based Testing
- Agile QA Practices

## Role Played
QA – responsible for end-to-end quality ownership
